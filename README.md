# 🌐 Nikita Saxena – React Portfolio

A modern, responsive developer portfolio built with **React + Vite** showcasing my projects, skills, and experience with an interactive UI and smooth animations.

---

## ✨ Features

🎨 **Sleek & Modern UI**  
A clean, visually appealing design with a professional aesthetic.

🌓 **Dark / Light Mode Toggle**  
Seamless theme switching that respects user preference and saves their choice.

🌌 **Interactive 3D Background**  
A captivating animated starfield background built with Three.js and React Three Fiber.

🎬 **Smooth Animations**  
Fluid page transitions and component animations powered by Framer Motion.

⏳ **Dynamic Experience Timeline**  
An interactive, scroll-animated timeline highlighting my professional journey.  
Details expand on **hover (desktop)** or **tap (mobile)**.

🖼️ **Project Showcase with Carousel**  
A grid of featured projects, each opening a modal with a swipeable image carousel.

📨 **Functional Contact Form**  
A working contact form integrated with EmailJS that sends messages directly to my inbox and provides an auto-reply to the sender.

📱 **Fully Responsive Design**  
Carefully designed to work perfectly on **mobile, tablet, and desktop**.

---

## 🛠️ Tech Stack

| Category | Technology |
|--------|-------------|
| Frontend | React + Vite |
| Styling | Tailwind CSS |
| Animation | Framer Motion |
| 3D Graphics | Three.js + React Three Fiber |
| Email Service | EmailJS |
| Deployment | Vercel |

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Nikita-Saxena391/nikita-saxena-portfolio.git
```

### 2️⃣ Navigate to the project folder

```bash
cd nikita-saxena-portfolio
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the development server

```bash
npm run dev
```

The app will run on:

```
http://localhost:5173
```
